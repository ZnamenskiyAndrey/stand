CRM

Это внутренняя система управленческого учета, интегрированная с API Федерального казначейства, позволяющая пользователям мониторить ход выполнения работ по национальному проекту, управлять списком задействованных при исполнении работ людей, поддерживать иные внутренние бизнес-процессы, возникающей в рамках исполнения национального проекта.

Портал

Также включает в себя публично доступную часть — портал берегдобрыхдел.рф, который отвечает за освещение хода работ, позволяет проводить сбор волонтеров, управлять распределением участников по этим работам, отслеживать ход их выполнения. Из функционала имеет API интеграцию с внутренней системой управленческого учета, медиа галерею, картографический модуль, систему генерации и выдачи дипломов волонтеров и другие, вполне стандартные для систем такого типа, модули.

В отдельных месяцах 2019–2020 гг. портал берегдобрыхдел.рф принимал до 120 000 посетителей в месяц
