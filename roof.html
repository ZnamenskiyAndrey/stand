Что делает

Подсчитывает раскрой материала для формирования уклонов и контруклонов для плоской кровли, учитывая расположение выходов на кровлю, водостоков, различных объектов находящихся на кровле.

Расчет производится с учетом различных факторов — размеров плит, особенностей крепежа, желаемого угла уклона и множества других.

Все эти параметры пользователь может указывать для конкретного расчета.

На вход сервис принимает чертеж крыши.

Предусмотрена простая версия интерфейса для создания чертежа в браузере

Калькулятор раскроя материалов

Инструменты предназначены для расчета раскроя материалов для создания уклона на плоской кровле, создания пошаговых инструкций поэлементной сборки.

Как считает

Алгоритм расчета описывается следующим образом:

  • 1

    Задаются границы крыши, местоположения сливов объектов на крыше

  • По сливам методом экстраполяции размечаются основные уклоны и их границы

    2
  • 3

    По границам уклонов производится раскладка плит разных типов для определения раскладки с наименьшими затратами по материалам

  • По карте уклонов подбираются параметры контруклонов в виде ромбов и их конфигурация для обеспечения минимального угла ската

    4
  • 5

    На полученных проекциях контруклонов производится раскладка материалов для определения раскладки с наименьшими затратами по материалам

  • Подсчитываются параметры контруклонов для обхода объектов на крыше

    6
  • 7

    Составляется карта высот для определения минимального угла уклона на всей крыше для проверки удовлетворения параметров

  • Составляется список плит и их форм в соответствии с порядком укладки

    8

Что делает кроме

  • Создает инструкции по укладке для монтажа;
  • Может планировать рез из плит разных размеров;
  • Дает рекомендации по палетированию нарезанных плит в нужном порядке;
  • Показывает укладку послойно;
  • Считает количество и стоимость крепежa;
